# Üniversite Şikayet Sistemi

Django tabanlı üniversite şikayet ve talep yönetim sistemi.

## Kurulum

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Özellikler
- Öğrenci kayıt/giriş (okul numarası ile)
- Şikayet oluşturma ve takip
- Yönetim paneli (durum değiştirme, silme)
- Şifremi unuttum
- Modern koyu tema arayüz
